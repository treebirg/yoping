#YOping

Command line client for YO

####Requirements: 
Node.js
npm

####Installation:

```sh
npm install yoping -g
```

OR

Download repo and
```sh
npm install -g
```


###Usage
Send YO to a user
```sh
yoping <username> 
yoping ping <username>
``` 
Update YO API key

```sh
yoping apikey <API_KEY>
```


Uses the yo-api package by sakatam: https://github.com/sakatam/node-yo-api
